thundermafia
============

GameJam AEEI-UQAC Automne 2013 - Équipe Thunder Mafia

Nom du jeu: **Dertal**


Thème et styles reçus
---------------------

Thème imposé: **L'infini et le néant**  
Styles imposés: **Shoot'em up** / **Rythmique**    


Touches
-------

* Déplacement: **W-A-S-D**
* Tir: **Clic droit**
* Rythme boss (debug): **Retour chariot**
* Retour normal (debug): **Effacement** 


Description du jeu
------------------

Dertal est un jeu dans lequel l'objectif est d'atteindre le plus haut score possible. Pour ce faire, il suffit de s'exiler le plus loin qu'on le peut de notre point de départ en sautant de dimensions en dimensions, qui sont représentées en 2D dans notre plan 3D, en suivant le rythme de la musique. À chaque "étage" on gagne un point.
On atteint l'écran de fin de partie lorsque notre barre de vie (représentée en vert) arrive à 0.
